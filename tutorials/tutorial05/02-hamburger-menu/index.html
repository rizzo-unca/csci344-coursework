This website has a hamburger menu!

The code is quite simple
we use some quick CSS tricks to make it so that when the website is in "mobile" mode
we take the nav bar and turn it into a hamburger menu!

All of the links in the nav bar work and will take you to the rest of the website as well as
back to the landing page for Tutorial 5.

To see the hamburger menu, you will need to shrink the website until it changes the nav
bar to the hamber menu icon of 3 -'s
